About SpendWise

SpendWise is a personal finance tracker that helps users manage income, expenses, savings, and investments through interactive charts and a clean, intuitive dashboard.

The application is built with Flask for the backend and Plotly for data visualizations. It supports user authentication and session management, with all financial data currently stored in a SQLite database.

Planned future upgrades include a transition to MySQL for scalability, the addition of downloadable reports, and expanded insights through goal tracking and trend detection.
